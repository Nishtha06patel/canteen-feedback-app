# 🍽️ Canteen Feedback App

A modern web-based feedback management system that enables students and customers to provide real-time feedback on canteen services. The platform helps canteen administrators analyze customer satisfaction, identify issues, and make data-driven improvements through interactive dashboards and analytics.

---

## 🚀 Key Features

### 👨‍🎓 User Features
- Submit canteen feedback and reviews
- Rate food quality and service
- Track submitted feedback
- Easy-to-use responsive interface

### 👨‍💼 Admin Features
- Centralized feedback management
- Real-time feedback monitoring
- Interactive analytics dashboard
- Visual reports and trends
- Data export functionality

### 📊 Analytics & Reporting
- Graphical representation of feedback data
- Performance tracking through charts
- Report generation and export options
- Trend analysis for better decision-making

### ⚡ Real-Time Updates
- Live data synchronization using Socket.IO
- Instant feedback visibility
- Improved responsiveness and engagement

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Nishtha06patel/canteen-feedback-app.git
cd canteen-feedback-app
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Run Backend Server

```bash
npm start
```

Traditional canteen feedback systems are often manual, inefficient, and difficult to analyze. This project provides a digital solution that streamlines feedback collection, enables real-time monitoring, and helps administrators improve service quality using actionable insights.
